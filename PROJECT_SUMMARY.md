# Project Summary: India Dashboard & Task Manager

## 1. Overview
A premium, full-stack style React application designed for managing clients, tasks, and team work sessions with a personalized Indian aesthetic.

## 2. Key Features
- **Premium UI/UX:** Glassmorphism, animated icons (Lucide/Framer Motion), and a custom India-themed background.
- **Multi-User Access:**
  - **Admin Mode:** Full control over clients, tasks, team members, and passwords.
  - **Team Mode:** Access to work sessions and task status updates.
  - **Guest Mode:** Read-only view of the dashboard.
- **Task Management:**
  - Support for **Weekly, Monthly, and Yearly** recurring tasks.
  - Real-time status tracking (Pending/Done).
- **Work Sessions:**
  - Start/End work tracking per client.
  - **Selfie Capture:** Integrated camera feature to take a selfie when starting work.
- **Reminder & Alarm System:**
  - Automated reminders at **10:30 AM**.
  - **Alarm Overlay:** Full-screen alert with sound for active tasks.
  - **Snooze/Dismiss:** Options to delay or clear reminders.
  - **Dismiss All (Admin Only):** One-click option to clear all pending reminders.
- **Security:**
  - Dynamic password system for Admin and Team logins.
  - LocalStorage persistence for all data.
- **Activity Logs:** Automatic logging of all major actions (Logins, Task completions, Client changes).

## 3. Technical Stack
- **Framework:** React 18 (Vite)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Data Persistence:** Browser LocalStorage (Scalable to Firebase/Backend)
